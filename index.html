How It Works
Two platforms, one simple flow
STEP 01
✍️
Fill Token Info
Set name, ticker, logo, description and social links. Supply is fixed at 1B.
STEP 02
⚙️
Configure Settings
Choose raised token (BNB/CAKE/USDT/USD1/ASTER/USDC), add tag, enable Antisnipe.
STEP 03
🚀
Launch & Buy
One transaction deploys the token and optionally buys from multiple wallets in the same block.
STEP 04
🎓
Auto-List on PancakeSwap
When bonding curve hits ~18 BNB, liquidity auto-seeds on PancakeSwap V2. LP burned forever.
STEP 01
✍️
Fill Token Info
Name, ticker, logo, description. Upload metadata — Flap stores it on IPFS automatically.
STEP 02
💰
Choose Token Type
Standard (no tax + FlapShare revenue) or Tax Token (0–10%, V2 pool, direct to beneficiary).
STEP 03
🦋
Launch via newTokenV2
BNBAgent calls Flap portal contract. Optional dev buy included in the launch transaction.
STEP 04
🎓
Auto-List PancakeSwap V3
At ~16 BNB, migrates to PancakeSwap V3. LP locked. FlapShare earns you fees every trade forever.
